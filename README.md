## 📌 Overview

Complete **Data Analysis Assignment** on **Bugatti Automobiles S.A.S.** — one of the world's most exclusive luxury hypercar manufacturers. Analyzes Bugatti's car lineup from **1924 to 2024**, covering horsepower trends, performance stats, price analysis, and visual data exploration.

## 📊 Dataset Features

| Column | Description |
|--------|-------------|
| `Model` | Car model name |
| `Year` | Year of production |
| `Engine` | Engine type |
| `Displacement_L` | Engine displacement in liters |
| `Cylinders` | Number of cylinders |
| `Horsepower_HP` | Power output in HP |
| `Torque_Nm` | Torque in Newton-meters |
| `Top_Speed_kmh` | Maximum speed in km/h |
| `Acceleration_0_100_s` | 0–100 km/h time in seconds |
| `Weight_kg` | Curb weight in kg |
| `Price_USD` | Price in US Dollars |
| `Country` | Country of manufacture |
| `Drive_Type` | AWD / RWD |
| `Transmission` | Gearbox type |
